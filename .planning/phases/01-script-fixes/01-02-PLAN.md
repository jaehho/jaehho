
This is the complete file. The only change from the current script is `LC_ALL=C ` prepended to `mpstat`. Do not change anything else. Preserve the shebang line exactly.

Why LC_ALL=C: When system locale uses comma as decimal separator (e.g., Fedora with LANG=fr_FR.UTF-8), mpstat outputs `12,50` for idle. AWK's numeric coercion reads `12,50` as the integer `12` (stops at comma), producing `100 - 12 = 88` instead of the correct `100 - 12.5 = 87`. LC_ALL=C forces mpstat to emit `12.50` (C locale dot-decimal) before awk sees the data.
